henzai

Native conversational AI for GNOME Shell using local LLMs

What is this

Local LLM integration for GNOME Shell. Press Super+H to open a chat panel, type a question, get streaming responses. Powered by Ramalama.

Install

Fedora 42/43:

sudo dnf copr enable csoriano/henzai
sudo dnf install henzai

Log out and back in. Press Super+H to use.
